Python入门

Python简介

脚本语言。解释语言。默认的解释器CPython。
Python之父—荷兰人Guido van Rossum。诞生于1989年。第一个公开发行版发行于1991年

Python用途

  1. 是网络应用，包括网站、后台服务等等；
  2. 其次是许多日常需要的小工具，包括系统管理员需要的脚本任务等等；
  3. 另外就是把其他语言开发的程序再包装起来，方便使用

Python优缺点

  • 性能差
  • 源码不能加密

Python 2.x vs 3.x

2008年, Python3发布。最大的变化是源码文件默认使用utf-8编码
